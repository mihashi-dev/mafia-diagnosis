# mafia-diagnosis (マフィア組織適性診断アプリ)

いくつかの心理テスト風の質問に答えることで、自分の隠れた性格や、もしマフィア組織にいたらどのポジション（ボス、参謀、特攻隊長など）に配属されるかを診断する動的Webアプリケーションです。

## 🚀 特徴
- **1ページ1問形式**: ユーザーがテンポよく回答できるストレスフリーな設計。
- **動的な結果画面**: Javaの条件分岐ロジックにより、回答の組み合わせに応じて診断結果がガラリと変化します。

## 🛠️ 使用技術 (Tech Stack)
- **Frontend**: HTML5, CSS3, JSP (JavaServer Pages)
- **Backend**: Java (Servlet 6.0 / Jakarta EE)
- **サーバー**: Apache Tomcat
- **データ管理**: HttpSession を利用したセッション状態の保持とスコア集計
